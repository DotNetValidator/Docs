DotNet Validator is a .NET library that contains a set of utility methods which could be used to validate or sanitize .NET data-models.

Installation

DotNet Validator is a portable .NET Standard library that could be used a cross all .NET platforms.

DotNet Validator 1.0.0 uses a .NET Standard 2.0 . See .NET Standard compatibility.

Install using:

  • Nuget Package Manager : Install-Package DotNetValidator -Version 1.0.0
  • .NET CLI : dotnet add package DotNetValidator --version 1.0.0
